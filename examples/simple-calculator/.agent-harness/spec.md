# Calculator Module

## Overview

Build a Python calculator module with comprehensive tests.

## Requirements

Implement these functions in `calculator.py`:

1. `add(a, b)` - Addition
2. `subtract(a, b)` - Subtraction
3. `multiply(a, b)` - Multiplication
4. `divide(a, b)` - Division (raise ValueError for division by zero)
5. `power(base, exp)` - Exponentiation
6. `modulo(a, b)` - Modulo (raise ValueError for modulo by zero)

## Technology Stack

- Python 3.x (stdlib only)
- unittest for testing

## Success Criteria

- All 6 functions implemented in `calculator.py`
- All tests pass in `test_calculator.py`
- Edge cases covered (zero, negative numbers, errors)
